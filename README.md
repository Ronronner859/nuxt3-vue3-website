# 前言

 本项目是一个nuxt3的企业级官网项目，开箱即用，直接下载依赖就可以运行。

**优势：**

1、采用vue3+Nuxt3最新技术，具有较高的网页渲染性能。

2、SSR渲染页面，有利于官网SEO，提高关键词在搜索引擎的排名。而且支持中英文、不同语种的SEO。

3、集成了**tailwindcss**、**vuetify**、**i18n**等优秀中间件。

4、屏幕尺寸自适应，无论是电脑、笔记本、平板、手机屏幕都能自适应显示，可一镜到底。
